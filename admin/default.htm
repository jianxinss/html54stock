HTML5开源股票行情图项目约定

HTML5开源行情图项目是非盈利的，对任何参与者都没有强性约束能力，但是大家有兴趣在一起做这件事情，需要一些约定才能把事情做好，所以我作为项目发起人起草下面的约定。

  1. 需求界定在HTML5行情方面，就是说我们只做行情方面的需求。其他需求，比如说游戏，可以另起项目,重新组织去做去，但不得在本项目中提需求
  2. 任何参与人都可以提出1中界定的需求
  3. 任何参与人都可以认领需求，认领需求时必须给出完成需求的dead line；尽可能随时更新进展情况
  4. 不同人可以认领相同的需求，最终采纳谁的作品，由大家投票决定
  5. 完成的代码量只要超过总代码量的十分之一就可以成为项目的Owner，文档可以按一定规则折合成代码行
  6. 除发起人之外，其他人都可以随时退出项目
